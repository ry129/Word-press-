== Description ==
##Easy share solution is Nice and awesome wordpress share plugin
Using Easy share solution plugin any wordpress site.Every website author want to use social share buttons.Easy share solution made for them.Easy share solution is ideal choice for start to create custom social share buttons and share counters.You can set share button 5 different position by this plugin. Easy share solution plugin also support share widget and text tweet.

This section describes how to install the plugin and get it working.

e.g.

1. Upload `Easy share solution.php` to the `/wp-content/plugins/` directory
